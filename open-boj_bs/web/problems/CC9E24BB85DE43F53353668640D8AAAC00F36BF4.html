Description

求解A*B，-1000 < A,B < 1000

Input

第一行一个数字t，代表有几组测试数据
接下来t行，每行两个数A和B

Output

对于每组测试数据，输出A*B

Sample Input

2
1 2
2 3

Sample Output

2
6

Source

Hint

这是另外一种输入的方式，而且这种方式正在逐渐的取代上面那种方式，因为有一个t的范围更容易让大家知道数据的规模。
下面是C++的样例:
#include
using namespace std;

int main(){
int a, b, t;
cin >> t;
while( t != 0 ){
t --;
cin >> a >> b;
cout << a * b << endl;
}

return 0;
}

接下来是ANSI C的样例：
#include

int main(){
int a, b, t;
scanf( "%d", &t );
while( t -- ){ <--------为什么可以这样写？思考一下:)
scanf( "%d%d", &a, &b)
printf( "%d\n", a * b);
}

return 0;
}
接下来是Java的样例：
import java.io.*;
