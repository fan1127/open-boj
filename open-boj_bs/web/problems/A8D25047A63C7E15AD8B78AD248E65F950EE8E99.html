Description

Many people know a sentence: "The quick brown fox jumps over the lazy dog". As we know, it contains 26 letters.
Now we want to know how many letters does a given sentence contain.

Input

First line contains a number t, indicate the number of test case.
Following t lines, each line contains a non-blankspace sentence.

Output

For each test case, output how many letters does it contain.

Sample Input

3
Thequickbrownfoxjumpsoverthelazydog
AbCdEfGhIjKlMnOpQrStUvWxYz
Just4test

Sample Output

26
26
5
