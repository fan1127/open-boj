Description

If n is a positive integer while it is not a square number, the continued fraction of its square root is interesting ---- sqrt(n) = ( a0, a1 a2 a3 ... am )
e.g. Sqrt(3)=

It can be represented as sqrt(3) = ( 1, 1 2 );
It can be confirmed that:

now input a natural number n, please output its continued fraction.

Input

First line contains a number t, indicate the number of test case.
Following t lines, each line contains a positive number n ( 0 < n < 10000 ).

Output

If n can NOT be expanded as a continued fraction, output "Error!"
If n can be expanded as a continued fraction, output its continued fraction as following format:
Output a0, then a dot and a blankspace.
Output following m number separated by a blankspace.
NOTICE: No blankspace after the last number.

Sample Input

4
3
4
13
30

Sample Output

1, 1 2
Error!
3, 1 1 1 1 6
5, 2 10

Source

Chen, Mingxia

Hint

Sqrt(x) means the Square root of x.
[x] is the max number that not larger than x.
