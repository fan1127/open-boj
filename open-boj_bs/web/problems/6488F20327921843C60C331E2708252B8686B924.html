Description

把字符串里所有大写字母变成小写字母。

Input

第一行一个数字t，代表有几组测试数据
接下来t行，每行一个字符串，字符串长度不超过30。

Output

对于每组测试数据，输出格式参照Sample Output，Case数从1开始。把字符串里的大写字母变成小写字母输出，

Sample Input

2
I'mLouty
I'mNotCrackerWang

Sample Output

Case 1: i'mlouty
Case 2: i'mnotcrackerwang

Source

Hint

这种输出方式在各大OJ上也是相当常见的，一旦人家让你按这样的格式输出，一定要注意Case的大小写，case数1，2是不是在递增，以及数字后面的冒号和冒号后的一个空格。错一点都会直接导致程序是Wrong Answer
