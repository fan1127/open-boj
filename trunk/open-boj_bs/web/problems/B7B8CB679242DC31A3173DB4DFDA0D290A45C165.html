Description

There is a match in bjtu's acm team. We have n ( 0 < n <= 100 ) members to take part in this match. All the people have a different speed s ( 0 < s <= 500 ). To make the result more immesurable, the runner will have different start-line. However they will have the equal end-line.
Louty is very fat, so Mr. Li want to know how many people finish the race after Louty.
You can assume no one finishing the race with Louty at the same time.

Input

First line contains a number t, indicate the number of test case.
For each test case:
Line 1: a number n indicate the members of the race.
Line 2: a positive number indicate the endline.
Line 3: Louty's speed and his starting-line.
Follow n-1 lines: each member's speed and starting-line.

Output

For each test case output a single number that how many people finish after Louty.

Sample Input

2
5
100
15 0
20 2
30 5
40 10
50 60
5
100
20 0
6 0
7 0
8 0
3 80

Sample Output


0
4
