Description

求解A+B，-1000 < A,B < 1000

Input

多组测试数据，每行两个数字A和B

Output

对于每组测试数据，输出A+B

Sample Input

3
5

Sample Output

1 2
2 3

Source

Hint

这是第一类的输入数据的类型，就是一直读输入文件直到EOF( End of File )
大家选择自己用的语言，仔细阅读一下下面的程序，不要直接用它提交AC，这样AC没有意义，因为以后的题你就不会了:(，磨刀不误砍柴工！

下面是C++的样例:
#include
using namespace std;

int main(){
int a, b;
while( cin >> a >> b )
cout << a + b << endl;

return 0;
}

接下来是ANSI C的样例：
#include

int main(){
int a, b;
while( scanf( "%d%d", &a, &b) != EOF )
printf( "%d\n", a + b);

return 0;
}
接下来是Java的样例：
import java.io.*;
import java.util.*;

public class Main
{
public static void main(String args[]) throws Exception
{
Scanner cin=new Scanner(System.in);

}
